# Walmart-Sales-Data-Analysis-Dashboard

## 📊 Project Overview
This dashboard project provides a comprehensive analysis of weekly sales data across 45 Walmart stores. The goal is to identify the key drivers of revenue, including seasonality, fuel prices, and the impact of holiday promotions.
**Dashboard Screenshot:** 'Walmart_Sales_Dashboard.png'

## 🛠️ Tools Used
**Excel:** Pivot Tables, Power Query, Slicers, and Advanced Charting.
**Conditional Formatting:** To highlight high performance stores.

## 💡 Key Insights
**Seasonality:** Contrary to typical retail trends, November contributes more to total revenue than the December holiday period.
**Economic Factor:** Fuel prices show a negative correlation with weekly sales; as fuel costs rose toward 2012, average consumer spending saw a measurable decline.

## 📂 How to View
1. Download the 'Walmart_Sales_Analysis.xlsx' file.
2. Open in Microsoft Excel to interact with the slicer.
